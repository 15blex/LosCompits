# Los Compits Website

This repository contains the source code for the Los Compits website, which is built using SvelteKit. The website serves as a platform to showcase the work and projects of Los Compits, a creative collective of musicians located in Barcelona.

The website features a clean and modern design, with a focus on user experience and accessibility. It includes sections for news, events, projects, and contact information, allowing visitors to easily navigate and learn more about Los Compits and their work.

## Technologies Used

- SvelteKit: A modern framework for building web applications with Svelte.
- Vanilla CSS: Used for styling the website, ensuring a responsive and visually appealing design.
- Typescript: Used for type safety and improved developer experience.
- Vite Plus: A fast build tool that provides a smooth development experience.
